Welcome to our API generator website, a project assigned by the esteemed company.

Our website is a powerful tool that can be used to scrape data from Yahoo! Finance of past 3 months and generate API endpoints that can be queried based on three parameters.

Firstly, our API can be queried based on a specific date.

All you need to do is input the date in the format of 'Apr 03, 2023', and our API will return all the companies' data for that particular date.



You can easily check this by typing the following
URL: https://hostaddress/stock-date/<DATE>,
where the example
URL: https://abc/stock-date/APR 03, 2023.
Secondly, you can query the API based on a company's name and date.

Input the company code name and date in the format of 'CSCO' and 'Apr 03, 2023,' respectively, and our API will return the particular company's data for that specific date.

You can check this by typing the following
URL: https://hostaddress/stock-date/<COMPANY CODE>/<DATE>,
where the example
URL: https://abc/s tock-date/CSCO/APR 03, 2023.
Lastly, you can query the API based on a company's name.

Input the company code name in the format of 'CSCO,' and our API will return all the data of that particular company.
You can check this by typing the following


URL: https://hostaddress/stock-date/<COMPANY CODE>,
where the example
URL: https://abc/stock-date/CSCO.



Our website also allows anyone to update the data of the database by sending a POST/PATCH update request to the website.

The sample URL format is
"http://127.0.0.1:8000/stock-update/<COMPANY CODE>/<DATE>,"
and an example URL is
"http://127.0.0.1:8000/stock-update/IBM/Apr 03, 2023."

However, please note that only existing rows in the database can be updated.


To add more company data, you can update config file. Our Application runs in the following sequence:
config.py -> assign.py -> app.py.

Lastly, our website continuously displays all the company names whose data is present, making it easy for you to find the data you need.

Our website is written using a combination of Python, Flask, SQLite, HTML, JavaScript, and Bootstrap, making it a versatile and powerful tool for anyone in need of up-to-date financial data. Thank you for choosing our website, and we hope you find it useful!
